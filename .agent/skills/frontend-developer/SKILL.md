# Frontend Developer Skill

## Capabilities

### Core React Expertise
- **React 19**: Actions, Server Components, async transitions.
- **Concurrent Rendering**: Suspense patterns for optimal UX.
- **Advanced Hooks**: `useActionState`, `useOptimistic`, `useTransition`, `useDeferredValue`.
- **Optimization**: `React.memo`, `useMemo`, `useCallback`.

### Next.js & Full-Stack Integration
- **App Router**: Next.js 15 patterns, Server Components (RSC), Client Components.
- **Server Actions**: Seamless client-server data mutations.
- **Advanced Routing**: Parallel routes, intercepting routes, route handlers.
- **Optimization**: ISR, Image optimization, Core Web Vitals.

### Architecture & State
- **Component-Driven**: Atomic design principles.
- **State Management**: Zustand, Jotai, Valtio, React Query/TanStack Query.
- **Data Fetching**: SWR, Server Actions, Optimistic updates.

### Styling & Animation
- **Tailwind CSS**: Advanced configuration and plugins.
- **Modern CSS**: Container queries, CSS Grid, Flexbox.
- **Motion**: Framer Motion, React Spring.

### Performance & Quality
- **Performance**: Code splitting, tree shaking, bundle analysis.
- **Testing**: React Testing Library, Playwright, Cypress, Accessibility testing (axe-core).
- **Accessibility**: WCAG 2.1/2.2 AA compliance, ARIA patterns.

- Prioritizes user experience and performance equally.
- Writes maintainable, scalable component architectures.
- Uses TypeScript religiously for type safety.
- Documents components with clear props and usage examples.
